HTML

  • HTML creates the structure of a website.
  • An element is a type of content on a webpage i.e. text, images, headers, and lists.
  • Most HTML elements start with an opening tag and end with a closing tag represented with angle brackets.
  • The head element contains information, metadata, that helps the browser to render the page correctly. This metadata contains data about the webpage that the browser uses to effectively serve the webpage to users. This contains important information about accessibility, search engines, and performance.
  • The body element contains the content that is shown to the user on the webpage.
  • The <!DOCTYPE html> declaration tells the browser it should expect HTML on the page.
  • An HTML attribute defines additional information about an element.
  • The title element defines the title of a webpage, which is important for Search Engine Optemization, or SEO.
  • The style element defines the CSS styles associated with a website.
  • The link element connects an external resource to the HTML document, such as an external CSS style sheet.
  • In semantic HTML elements, the name of the element describes its content and the role it plays on a webpage. This makes the HTML easier to read by other developers and the browser. Additionally, semantic HTML makes the webpage accessible to those who use assistive technologies such as screen readers.
  • Section elements organize a webpage into sections by thematic content, such as HTML, CSS, Git, and JavaScript.
  • The div element creates a division on the page.
  • The span element creates an inline division on the page.
  • The h1, h2, h3, h4, h5, and h6 elements represent the level of heading a given text block represents. The lower the number, the larger and more prominent the heading.
  • The p element represents a paragraph or block of text.
  • Th ul element creates an unordered list, the ol element creates an ordered list, and the li element creates a list item.
  • The img element contains information about images that are displayed on the webpage.
  • The scr attribute defines the location of the image.
  • The br element creates a line of empty space, or a line break, between two blocks of content.
  • The anchor element creates links to the same webpage or other webpages.
  • The href attribute points to the URL of the link.

CSS

  • CSS adds style to the webpage. It defines the colors, aesthetics, visual layout, and organization. We can use CSS to define style rules for specific elements of an HTML page and format them using a wide ranfe of style options.
  • Inline CSS is used to style a specific HTML tag using the style attribute with a CSS style rule. This is for internal styling, within the HTML document.
  • An external style sheet, which allows developers to keep all their CSS rules in a separate file, making the design changes easier.
  • A selector defines the element or attributes to which the rules, or declarations, will apply.
  • Declarations contain the CSS property we want to apply to the element and the value of the property.
  • We can think about each HTML element as a box. The CSS box model refers to the margins outside the element and the element's borders, padding, and content.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • To apply a rule to all elements visible on the page, we use an asterisk.
  • We can apply CSS rules to certain elements as well by using a comma between the two elements. We can center an element on a webpage by using the text-align property and applying the value of center.
  • The display property with the value of block assigns an element to start on a new line.
  • To link the external style sheet to the HTML document, we use the link element with the rel and href attributes. The rel attribute is assigned “stylesheet” and the href attribute is assigned the location of the style sheet within our folder.
  • We can use class attributes as selectors which allows us to share a CSS rule to any element we choose by assigning the rule to a class attribute with a class selector. The class attribute can be assigned to the element in the HTML, enabling us to freely choose with elements will be assigned the CSS rule.
  • We can use the card class selector, which applies the CSS rules for any HTML element with the class of card. We put a period preceding card, which designates this as a class selector.

Git

  • Git is a version control system that lets you manage and keep track of each version of your project. It helps us track changes locally on our computer.
  • Git is accessed through the command line. These commands always start with git.
  • Feature branches are where we add any new code.
  • The default branch is called main, or sometimes master. No new code should be added to this branch.
  • Once the changes from the feature branch are approved, they are merged into the main branch.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: adds all changes to code. This command tells Git that the changes will be added in the next commit.
  • git commit -m "message": stages the files to be committed. Once committed, the feature branch will be updated with the new code.
  • git pull origin main: makes sure that we are working on up to date code. This makes sure our local branch is in sync with the main branch in GitHub.
  • git push origin branch-name: pushes the changes to the remote branch.

JavaScript

  • JavaScript takes a static HTML/CSS webpage and makes it interactive. For example, this allows the user to click a button, retrieve data, or fill out a form.
  • Each line of JavaScript code ends with a semicolon.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • One type of conditional statement in JavaScript is an if statement. An if statement is a way of introducing decision-making into our code. Code will only execute if the statement meets a condition, or if it is true. If the computer encounters a truthy if statement, it will do whatever the action we ask it to do in between the curly brackets. If the condition is not true or has a value that is not truthy, the computer will ignore whatever is in between the curly brackets and move on.
  • Another conditional statement is an else if statement. This allows us to provide more than one condition to our if statements. The computer will go through all the conditions listed, evaluating whether or not a condition is truthy. Once the computer finds a truthy condition, it will run the code in the curly brackets associated with that condition and break out of the if statement. The else statement provides the computer with a statement to log to the console if none of the else if statements are truthy.
  • We use the equality operator === to check if two things are equal. If they are equal, it returns a Boolean result true. If they are not equal, it returns a Boolean false.
  • An array is a single variable that is used to hold a group of data. They are typically used to hold data that is related in some way. We use brackets to store arrays.
  • To access a certain data item within an array, we use the location of the data item in the array that is identified with a unique number called an index. The first item is always starts from 0 and so on. To access the item, we name the variable and put the location identifier in brackets.
  • A for loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • To comment out code in JavaScript, we put the code between /* and */.
  • A function is a set of instructions that tells the computer how to perform a certain task. Functions do not automatically execute when the JavaScript file is run. In order for it to run, we need to call it by name. This makes the code easily reusable because we can call it as many times as we need.
  • The Math function Math.random() helps us select a random number.
  • We use console.log(); to output a message to the web console by adding an argument in the parentheses.
  • Chrome DevTools allows us to see our console.log(); output.
